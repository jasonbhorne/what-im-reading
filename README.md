# What I'm Reading

The data source for the [What I'm Reading](https://jasonhorne.org) page on jasonhorne.org.

`index.html` shows my current Audible reading list and shelf. It is regenerated
by the `/what-im-reading` Claude Code skill (a live Audible pull) and served via
GitHub Pages. The live jasonhorne.org page fetches `index.html` on load, so the
website updates itself with no manual editing.

Built and maintained with Claude Code.
